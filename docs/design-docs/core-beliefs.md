# Core Beliefs

- The site should feel premium, editorial, and sharp.
- The design should support trust and conversion, not decoration for its own sake.
- Real work, real service clarity, and strong CTAs matter more than ornamental UI.
- Performance and SEO are part of the product promise.
- `pt-BR` copy must be polished and correctly accented.
- Local fonts, existing tokens, and Material Symbols keep the design system stable.
